# Harvey — Series A Pitch Deck Demo

[中文版](README.zh-CN.md)

A 19-slide Series A pitch deck generated by `founder-pitch-deck` in the style of [Harvey AI](https://harvey.ai) — the $11B legal AI company.

## How it was made

```
/founder-pitch-deck source=https://harvey.ai
```

The skill extracted Harvey's dark, editorial design language by screenshotting the reference deck (17 slides), then:

1. **Design extraction** — Cormorant Garamond serif + Inter sans-serif, dark `#0F0E0D` / light `#FAFAF9` alternation, gold accent `#B8956A`
2. **Content** — based on publicly available data (TechCrunch, company blog, SEC filings)
3. **Narrative review** — YC Office Hours critique on all 19 slides
4. **HTML generation** — single-file `index.html`, no external dependencies

## View it

```bash
open index.html
# Password: harvey
```

## What to look for

- **Auth gate** — password-protected entry (type `harvey`)
- **Bilingual toggle** — click ZH/EN button in the corner
- **Dark/light alternation** — slides alternate between `#0F0E0D` and `#FAFAF9`
- **Typography** — Cormorant Garamond headings, Inter body, weight 500 headings (never bold)
- **Stat counters** — animated count-up numbers on data slides
- **Staggered reveals** — bullets and cards appear with 120ms offset
- **PDF export** — Ctrl+P or click the PDF button (watermark auto-applies)
- **Responsive** — resize the window, it scales proportionally
